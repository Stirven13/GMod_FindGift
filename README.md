## Установка
Вам необходимо скачать [архив](https://github.com/Stirven13/GMod_FindGift/archive/refs/heads/main.zip).

Из этого архива нужно будет перекинуть файл `gift_finder.lua` в `...\Steam\steamapps\common\GarrysMod\garrysmod\lua`.

## Запуск
Для запуска нужно ввести в консоль `lua_openscript_cl gift_finder.lua`

## Остановка
Для остановки можно ввести в консоль `giftoff` или `offgift`.

## Удаление
Чтобы полностью стереть скрипт нужно:
1. Выключить его прописав `giftoff` в консоль.
2. Удалить файл `gift_finder.lua` из `...\Steam\steamapps\common\GarrysMod\garrysmod\lua`.
3. Очистить Корзину, либо удалить файл `gift_finder.lua` из Корзины.
4. Удалить zip-файл из Загрузки (`C:\Users\<UserName>\Downloads`).
5. Очистить историю браузера.
